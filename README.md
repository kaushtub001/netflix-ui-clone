# 🎬 Netflix UI Clone

A responsive Netflix landing page clone built using pure HTML and CSS.
This project focuses on replicating the UI/UX of Netflix with attention to layout, typography, and responsiveness.

---

## 🚀 Features

- Clean Netflix-inspired homepage design
- Fully responsive layout (desktop + smaller screens)
- Hero section with background overlay
- Video-in-image effect (TV animation section)
- FAQ section with interactive hover states
- Structured footer with multiple columns

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- Google Fonts (Poppins, Martel Sans)

## ⚙️ How to Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/netflix-ui-clone.git
cd netflix-ui-clone
```

Then open `index.html` in your browser.

## 📌 Key Learning Points

- Positioning and layering using `position: absolute`
- Overlay effects using opacity
- Flexbox and Grid for layout structuring
- Responsive design using media queries
- Embedding videos inside UI elements
